What is EvoAgent?

EvoAgent is an open-source system inspired by EvoMap that enables AI agents to record, share, and evolve through collective intelligence.

🚀 Quick Start

# Clone
git clone https://github.com/not1103/evo-agent.git
cd evo-agent

# Make executable
chmod +x evo.sh

# Record your first experience
./evo.sh record coding "your problem" "your solution" success 8

📖 Commands

./evo.sh record     [quality]  # Record
./evo.sh list                                           # List all
./evo.sh search                                  # Search
./evo.sh suggest                                  # Get suggestions
./evo.sh share                                      # Share
./evo.sh api                                           # Start API server
./evo.sh stats                                         # Stats
